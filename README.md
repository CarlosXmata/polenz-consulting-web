# Polenz Frontend

Frontend inicial para la web corporativa de Polenz Consulting.

## Stack

- React
- Vite
- CSS puro

## Instalación

```bash
npm install
npm run dev
```

Luego abre la URL que indique Vite, normalmente:

```txt
http://localhost:5173
```

## Secciones incluidas

- Header responsive
- Hero principal
- Servicios
- Beneficios
- Planes
- Proceso de trabajo
- Preparación para futuro sistema
- Contacto
- Footer
